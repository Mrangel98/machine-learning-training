# Machine Learning Training

Repositorio con distintos ejercicios y experimentos de Machine Learning desarrollados en Python y R.

## Tecnologías utilizadas

- Python
- Scikit-Learn
- PyTorch
- TensorFlow
- R

## Proyectos incluidos

### 1. Regresión y clasificación
Implementación de modelos de machine learning utilizando Scikit-Learn para resolver problemas de regresión y clasificación.

### 2. Red neuronal MLP
Implementación de una red neuronal multicapa (MLP) para tareas de predicción.

### 3. Deep Learning con PyTorch
Ejemplo de entrenamiento de una red neuronal utilizando PyTorch.

### 4. Deep Learning con TensorFlow
Implementación de un modelo de deep learning utilizando TensorFlow.

### 5. Análisis de datos en R
Script de análisis de datos y modelado estadístico utilizando R.
